Inside of ./Putnam_Source_Files/ there is a script called ./PutnamExams/get_putnam_all.sh
Run this script within that folder and it will download all the putnam exams 1985-2022 and all solutions 1995-2022.

Then, compile the LaTeX file PutnamExams/Putnam.tex and you will have a single pdf file with all previous putnam exams in it.
Similarly, PutnamSolutions/PutnameSolutions.tex will yeild single pdf with all the solutions in it.
